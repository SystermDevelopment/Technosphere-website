# Docker環境

## 実行方法

1. Dockerをインストール
   [Get Docker | Docker Docs](https://docs.docker.com/get-started/get-docker/)

2. 起動
    ```shell
    cd .docker
    bash startup.sh # (linux)
    # or `docker compose up -d` (windows, mac)
    ```
